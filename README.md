ProActiva Voice Assistant

A browser-based voice assistant web application that enables real-time, hands-free interaction through speech. Built with HTML, CSS, JavaScript, Node.js, and SQLite, ProActiva integrates Web Speech APIs for voice recognition and synthesis, delivering both utility and entertainment features in a futuristic UI.

Features

Voice Recognition & Speech Synthesis
 -Captures spoken commands and responds audibly.
 -Supports greetings, time/date queries, rhymes, jokes, and more.
Website Navigation & Search
 -Commands like “Open YouTube” or “Search for sneakers” generate dynamic redirect buttons.
Interaction History
 -Logs all commands and responses in SQLite.
 -Provides a scrollable history panel.
 -Download history as a .txt file for offline reference.
Dynamic UI
 -Blurred glass effects, glowing buttons, and responsive design.
 -Background images change automatically every 5 seconds.
 -Theme toggle (light/dark mode) for visual comfort.

Tech Stack

-HTML	    --->           Structure of the web application.
-CSS	    --->           Styling, animations, glowing buttons.
-JavaScript	 --->        Core logic, voice recognition & synthesis.
-Web Speech API	 --->    SpeechRecognition & SpeechSynthesis.
-Node.js	   --->        Backend runtime.
-SQLite	      --->       Database for storing interaction history.
-Fetch & Blob	 --->      Data transfer and history download.

Project Modules

Speech Recognition Module – Captures and converts voice input to text.
Command Processing Module – Interprets commands and triggers actions.
Speech Synthesis Module – Provides audible responses.
UI Module – Manages dynamic backgrounds, animations, and theme toggle.
Database Module – Stores user interactions in SQLite.
Download History Module – Exports interaction logs as .txt files.

Conclusion

ProActiva Voice Assistant is a lightweight, browser-based application that combines speech recognition, speech synthesis, dynamic UI, and database logging to deliver a seamless hands-free experience. 
It enables users to interact through voice for tasks like website navigation, searches, and fun educational commands, while offering features such as downloadable history and dynamic backgrounds. 
The project demonstrates the effective use of modern web technologies and sets the stage for future enhancements like NLP, multi-language support, and mobile app integration.
